如何处理博客图片

0x00 图片压缩

由于不放心自动压缩后上传，所以使用这个脚本压缩一下图片。检查后再上传图片。
使用的是Tinypng 这个网站的API，需要在线申请一个Key，然后每个月可以压缩500张图片。
接着使用这个repo 的python 脚本 TinyPng。
python tinypng.py -d test
这样就压缩了test目录下的所有图片。
